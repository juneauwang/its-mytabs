# 国际化 (i18n) 支持

## 概述

整个应用已实现完整的中英文双语支持。用户可以通过页面底部的切换按钮在 **简体中文** 和 **English** 之间切换。

## 使用方法

1. 滚动到任意页面底部
2. 点击 **中文** 或 **English** 按钮
3. 页面自动刷新，语言即时切换

语言偏好保存在浏览器 `localStorage` 中，下次访问时自动使用上次选择的语言。

## 技术实现

### 框架

使用 [vue-i18n](https://vue-i18n.intlify.dev/) v9 作为国际化框架。

### 文件结构

| 文件 | 说明 |
|------|------|
| `frontend/src/i18n.ts` | i18n 配置，注册支持的语言 |
| `frontend/src/lang/en.json` | 英文翻译（约 200 个键） |
| `frontend/src/lang/zh-CN.json` | 简体中文翻译（约 200 个键） |
| `frontend/src/layouts/Layout.vue` | 底部语言切换按钮 |

### 翻译键命名空间

| 命名空间 | 对应页面 |
|----------|----------|
| `common.*` | 通用文字（保存、删除、上传、是/否等） |
| `editor.*` | NoteEditor 编辑面板 |
| `tab.*` | Tab.vue 播放器页面 |
| `tabConfig.*` | TabConfig.vue 配置页面 |
| `settings.*` | Settings.vue 设置页面 |
| `home.*` | Home.vue 首页 |
| `dashboard.*` | Dashboard.vue 仪表盘 |
| `auth.*` | Login.vue / Register.vue 登录注册 |
| `tabNew.*` | TabNew.vue 新建谱面 |

### 添加新语言

1. 在 `frontend/src/lang/` 下创建翻译文件（如 `ja.json`）
2. 在 `frontend/src/i18n.ts` 中导入并注册
3. 在 `languageList` 中添加语言名称
4. 在 `Layout.vue` 的语言切换按钮中添加新按钮
